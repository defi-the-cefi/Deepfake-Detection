### Usage

  * git clone this repo

  * cd into repo directory

  * create conda env

  ```
  conda env create -f env.yml
  ```

  * unzip images for classification to ./deepfake_detection/dataset folder such that their are 2 subdirectories "real" and "fake"
  ```
  cp /image/source/path ./images
  ```
